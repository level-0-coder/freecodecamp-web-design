Introduction

Markdown is a lightweight language, and it’s not a programming language

It’s used to format text, making it look nice and readable. You can use it to create web pages, documents, or even emails.

Unlike HTML which is a markup language with lots of customization when combined with css, markdown doesn't have that many customization options. But it's enough to distinguish and prettify the notes(by devs).

Where and why should you use it?

Here are the main reasons why markdown is used.

  • It's quicker to format the text once which helpful in creating documentations with many pages
  • It's relatively simpler to use than the html and css because of less syntax
  • Also note that less and short syntax means less customization or formatting options
Heading in markdown

Headings are used to display titles and topics

There are 6 headings available in markdown each with unique font size

# h1
## h2
### h3
#### h4
##### h5
###### h6
Lists in markdown

We can create bullet lists in markdown using "-" and a space followed by the content.

- item1
- item2
- item3
- and so on...
Font styles in markdown

We can format the given to bold, italics, and highlight

  • bold: "**text**" is used to make text bold
    text in **bold**
  • italics: "*text*" is used to make text italic
    text in *italics*
  • highlight: "==text==" is used to highlight a text
    ==highlighted text==
Block quote in markdown

Block quotes make the text more distinct and clear and it can be written using the below syntax

> text after ">" and a space is rendered as blockquote
